Highlighted Projects

1. EC2 Auto Start/Stop with Lambda

Problem: EC2 instances were running 24/7, increasing costs.
Solution: Implemented AWS Lambda with CloudWatch Events to start/stop instances on schedule.
Outcome: Reduced costs by ~40%.

2. Real-time Monitoring with Grafana, Prometheus & Loki

Problem: Lack of centralized monitoring and alerting.
Solution: Built dashboards using Grafana, Prometheus, Loki, Promtail, Blackbox Exporter, and Node Exporter.
Outcome: Improved observability, reduced downtime.

3. Automated Static Website Deployment

Problem: Manual deployment process for websites.
Solution: Automated using AWS S3, CloudFormation, and CI/CD workflows.
Outcome: Faster and reliable deployments.

4. High-Security Infrastructure Migration

Problem: Needed migration with strict security compliance.
Solution: Migrated firewalls, web servers, and databases with hardened security.
Outcome: Zero security breaches, smooth transition.

5. AWS Architectures with Subnets, VPN, Load Balancers, RDS

Problem: Required scalable, hybrid architecture.
Solution: Designed VPC with private/public subnets, VPN connectivity, load balancers, and RDS (MySQL, MongoDB via OpenVPN).
Outcome: Secure and scalable infrastructure.

6. ECS Fargate & CI/CD Pipelines

Problem: Need for containerized, auto-scalable deployments.
Solution: Built infra on AWS ECS Fargate integrated with CodePipeline CI/CD.
Outcome: Increased deployment speed, reduced manual intervention.
