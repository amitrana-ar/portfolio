Blue-Green Deployment on ECS (0% Downtime) with CodePipeline CI/CD

Developed a fully automated Blue-Green Deployment solution for AWS ECS services with zero downtime during production releases.

When developers push new code to the repository, AWS CodePipeline automatically triggers the build process via CodeBuild. It builds a Docker image, pushes it to Amazon ECR, and deploys it using an ECS Blue-Green deployment strategy that updates ECS services seamlessly.

This setup improved deployment reliability, eliminated downtime, and enabled safe, automated rollback mechanisms for production systems.
